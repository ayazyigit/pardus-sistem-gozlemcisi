# Kurulum Adımları;
1. Konsolu açınız.
2. Konsola "cd Masaüstü" komutunu giriniz.
3. Konsola "git clone https://github.com/ayazyigit/pardus-sistem-gozlemcisi" komutunu giriniz.
4. Masaüstünde oluşan "pardus-sistem-gozlemcisi" klasörüne giriniz.
5. Klasörün içindeki "pardus-sistem-gozlemcisi-paket.deb" dosyasını açınız.
6. "Yükle" tuşuna basınız.
7. Başlatma menüsünden "Pardus Sistem Gözlemcisi" uygulamasını aratınız.

# Kaldırma Adımları;
1. Pardus Yazılım Merkezi'ne girin.
2. Üstten depo sayfasına girin.
3. Depo sayfasındayken "pardus-sistem-gozlemcisi" deposunu aratın.
4. Çıkan ilk depoya basın.
5. Sağ tarafta çıkan kırmızı "Kaldır" butonuna basın.

**Ayaz YİĞİT** tarafından yapılmıştır. İzinsiz çoğaltılması ***yasaktır***.
